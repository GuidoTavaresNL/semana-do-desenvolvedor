⚡ Código Lambda – Pré-validação

Runtime: Python 3.12 | Variável de ambiente: SQS_QUEUE_URL

python – pre-validacao-lambda 
import json, os, boto3

SQS_QUEUE_URL = os.environ['SQS_QUEUE_URL']
sqs = boto3.client('sqs')

def lambda_handler(event, context):
    body = json.loads(event.get('body', '{}'))
    pedido_id = body.get('pedidoId')
    cliente_id = body.get('clienteId')

    if not pedido_id or not cliente_id:
        return {'statusCode': 400, 'body': json.dumps({'message': 'pedidoId e clienteId são obrigatórios'})}

    response = sqs.send_message(
        QueueUrl=SQS_QUEUE_URL,
        MessageBody=json.dumps(body),
        MessageGroupId=str(pedido_id),
        MessageDeduplicationId=str(context.aws_request_id)
    )
    return {'statusCode': 200, 'body': json.dumps({'message': 'Pedido enfileirado', 'sqsMessageId': response['MessageId']})}

⚡ Código Lambda – Processamento Central

python – processa-pedidos-lambda
import json, os, boto3
from datetime import datetime

DYNAMODB_TABLE_NAME = os.environ['DYNAMODB_TABLE_NAME']
dynamodb = boto3.resource('dynamodb')
table = dynamodb.Table(DYNAMODB_TABLE_NAME)

def lambda_handler(event, context):
    for record in event['Records']:
        eventbridge_event = json.loads(record['body'])
        pedido_data = eventbridge_event['detail']
        pedido_id = pedido_data.get('pedidoId')

        item = {
            'pedidoId': str(pedido_id),
            'clienteId': str(pedido_data.get('clienteId')),
            'itens': pedido_data.get('itens', []),
            'statusPedido': 'PEDIDO_PROCESSADO',
            'origem': pedido_data.get('origem', 'API'),
            'timestampProcessamento': datetime.utcnow().isoformat() + "Z"
        }
        table.put_item(Item={k: v for k, v in item.items() if v is not None})
        print(f"Pedido {pedido_id} salvo com status PEDIDO_PROCESSADO")

    return {'statusCode': 200, 'body': 'Processamento concluído'}

Envie um novo pedido pela API

3
Verifique a DLQ pedidos-pendentes-dlq

4
Remova o erro e restaure

Volte à Lambda, remova o raise ValueError e clique em Deploy novamente.
